Comment utiliser les colonnes ?

Pour utiliser le système de colonnes, il faut toujours encapsuler un set de colonnes dans une div avec la class : .columns

Ensuite, vous pouvez jouer avec les combinaisons suivante :

  • .whole : définit une colonne qui prend toute la largeur de son parent (et bénéficie d'un clear:both)
  • .one-half : définit une colonne 1/2
  • .one-third : définit une colonne 1/3
  • .two-third : définit une colonne 2/3
  • .one-fourth : définit une colonne 1/4
  • .three-fourth : définit une colonne 3/4
