Les boutons

Il est possible de créer plusieurs tailles de bouton :

Bouton Big Bouton "Big" généré grâce à la class : .a-btn-big

Bouton Small Bouton "Small" généré grâce à la class : .a-btn-small

Les séparations avec la balise hr

Il est possible de créer deux types de séparation :

  • Une séparation par un trait en utilisant la balise hr
  • Une séparation invisible en utilisant la balise hr avec la class : .invisible
